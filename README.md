# Rev Workforce

Rev Workforce is a console-based Human Resource Management (HRM) application designed to efficiently manage employee-related operations such as employee records, leave management, and performance evaluation.

The system supports three primary user roles: Employees, Managers, and Admins, each with role-specific functionalities to ensure proper access control and workflow management.

Employees can manage their profiles and request leaves, Managers can review team performance and approve requests, while Admins have full control over system operations including user and role management.

The application follows a modular and layered architecture, making it scalable and maintainable. 

This project demonstrates core concepts of Java, object-oriented design, layered architecture.

##  Tech Stack

- **Java** – Core application development  
- **JDBC** – Database connectivity  
- **MySQL** – Relational database  
- **Log4j** – Logging framework  
- **JUnit** – Unit testing
